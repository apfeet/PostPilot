# PostPilot

## ⚙️ Dependencies

- Docker installed 🐳

## 🚀 How to Run the Project

1. Rename the file `.env.example` to `.env` and populate with appropriate values.

2. Run the project: ` docker-compose up --build`
   
## 🌐 Access Pages
FrontEnd: http://localhost:5173 🌟\
Backend: http://localhost:5000 🔧
